The project was about making a clone of Moodle, the university's learning management system, using Angular for the frontend and Django for the backend. Leveraging Angular's capabilities, the frontend was designed to provide a user-friendly interface for students and instructors to access course materials. On the backend, Django facilitated the development of a robust and scalable platform, handling user authentication and course management.

The project involved creating a clone of Instagram using Angular for the frontend and Spring Boot for the backend. The Angular framework was utilized to develop the user interface, providing a sleek and intuitive design reminiscent of the original Instagram platform. Meanwhile, Spring Boot served as the backend framework, facilitating the development of robust and scalable RESTful APIs to handle user authentication, image uploading, commenting, liking, and other core functionalities.

The project was about developing a software application capable of extracting lyrics from an MP3 file of a song and displaying them elegantly line by line on the screen. Through sophisticated audio processing techniques, the application analyzed the audio waveform of the MP3 file to identify and extract the lyrics embedded within the song. The extracted lyrics were then dynamically presented in a visually appealing manner, with each line synchronized to the corresponding portion of the song.
